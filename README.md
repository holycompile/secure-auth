# 🔐 Cypher - Secure Access System

A simple Authentication and Access Control system built using Flask.

---

## 🚀 Features
- User Registration & Login
- Password Hashing using bcrypt
- Session Management
- Role-Based Access Control (Admin/User)
- Protected Dashboard
- Admin Panel with Access Restriction

---

## 🧭 How to Use
1. Register a new user
2. Login using credentials
3. Access your dashboard
4. Try accessing admin panel
5. Logout and retry access

---

## ⚠️ Limitations
- No Multi-Factor Authentication (MFA)
- No brute-force protection
- Uses SQLite (not scalable for production)

---

## 🛠 Tech Stack
- Python (Flask)
- SQLite
- HTML, CSS

---

## 📌 Project Purpose
To demonstrate core concepts of authentication and access control in web systems.

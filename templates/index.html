Cypher Security

The Simple Security System

A lightweight authentication and access control system designed to demonstrate secure login, role-based access, and session management.

🧭 Instructions

1️⃣ Register
Create a new account using the Register page with your username and password.
2️⃣ Login
Use your registered credentials to log into the system securely.
3️⃣ Dashboard
After login, you will be redirected to your dashboard based on your role (User/Admin).
4️⃣ Admin Access
Attempt to access the Admin panel. Only admin users will be allowed entry.
5️⃣ Logout
Logout and try accessing protected pages again to verify active session security.

📌 About This Project

This project demonstrates how authentication, session control, and access levels operate. Users can register, login securely, and access resources based on their assigned roles, showing robust, clean development.

⚙️ Features

🔐 Secure Password Hashing
Utilizes bcrypt with salt generation to securely store credentials.
👥 Role-Based Access
Separates privileges between general Users and System Administrators.
🧠 Session Management
Tracks user logins via secure cookie-based session states.

⚠️ Limitations & Improvements

🔒 Basic Authentication
Uses basic user/password auth. In this version, we've integrated **password updates** from the dashboard.
🛡️ Security Logs & Limits
Now includes **brute-force protection** (locks user attempts after 5 failures) and **login logs**.
👥 Admin Control Panel
Features a new **User Management Dashboard** allowing admins to promote/demote roles or delete users.

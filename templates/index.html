The Simple Security System

A lightweight authentication and access control system designed to demonstrate secure login, role-based access, and session management.

🧭 Instructions

1️⃣ Register
Create a new account using the Register page with your username and password.
2️⃣ Login
Use your registered credentials to log into the system.
3️⃣ Access Dashboard
After login, you will be redirected to your dashboard based on your role.
4️⃣ Try Admin Access
Attempt to access the Admin panel. Only admin users will be allowed.
5️⃣ Logout
Logout and try accessing protected pages again to verify security.

📌 About This Project

This project demonstrates how authentication and access control work in a system. Users can register, login securely, and access resources based on their roles.

⚙️ Features

🔐 Secure Password Hashing (bcrypt)
👥 Role-Based Access (Admin/User)
🧠 Session Management
🚪 Login & Logout System

🛠 Built With

🐍 Python (Flask)
🗄 SQLite Database
🎨 HTML + CSS

⚠️ Limitations of This Project

🔒 Basic Authentication Only
This project uses simple username-password authentication and does not include advanced methods like OAuth, Multi-Factor Authentication (MFA), or biometrics. Same user cannot change password, create new passwords or delete account.
🛡️ Limited Security Features
While passwords are hashed, the system does not include protections like rate limiting, CAPTCHA, or brute-force attack prevention.
👥 No User Management System
Admin cannot currently manage users (edit roles, delete users, etc.).
🌐 Not Production Ready
This is a learning/demo project and lacks deployment-level security configurations.
📦 Uses SQLite Only
SQLite is suitable for small-scale apps but not ideal for large, scalable systems.
